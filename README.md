## Keybinds

 <ul>
  <li> Insert csv file(Ctrl+O)</li>
  <li> Save csv file(Ctrl+S) </li>
  <li> Exit(Alt+F4) </li>
  <li> Close csv(Ctrl+F5) </li>
  <li> Delete duplicates(Ctrl+D) </li>
  <li> Drop Non Values(Ctrl+W) </li>
  <li> Fix Non Values(Ctrl+F)</li>
  <li> About(Ctrl+I)</li>
  <li> Help(Ctrl+F1)</li>
</ul>
